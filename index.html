Mirabel Wohler SWR MODFLOW Model Results

	These are model runs from the SWR MODFLOW model of
	the Mirabel Wohler Area. The Model is under development.
	Each Run reflects the incoming stream inflows
	from Mark West Creek, Dry Creek, and the Russian River.

	The dam elevations are interpreted from the RDS stage and other data.

	Initial conditions: model run 3 times for 10 days using
	observed flow/pumping/dam elevation for model periods.
	Heads are recursively used as initial conditions.
			
Other factors that need to be addressed:
	-Diversions from ponded area. Currently recharged water magically arrives at ponds.
